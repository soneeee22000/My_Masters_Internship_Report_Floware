# 🎓 Internship Report – Cloud Data Batch Processing at Floware SAS

Welcome to the official repository of my internship report titled  
**"Floware BP – Setting Up Cloud Data Batch Processing Infrastructure Using Microsoft Azure Batch"**.

This report marks the culmination of my 6-month internship at **Floware SAS**, a deeptech startup based in France, and the completion of my **Dual Master's Degree in Data Science, Artificial Intelligence & Network Intelligence** from the **Asian Institute of Technology (AIT)** and **Télécom SudParis (TSP)**.

## 📘 About the Report

The internship focused on designing and deploying a scalable, automated cloud data processing pipeline using **Microsoft Azure Batch**, **Docker**, and **Python**. The project aimed to transform Floware’s manual data workflows into a robust cloud-native architecture capable of handling large-scale sensor data from mobility detection systems.

Key achievements include:
- ✅ End-to-end Azure Batch pipeline setup with containerized tasks
- 📦 Integration of Azure Blob Storage, Container Registry, and Function Apps
- 🧠 Automation of Computer Vision and Bluetooth trajectory data processing
- 📊 Development of PowerBI dashboards for client-facing analytics
- 🔧 Hands-on deployment of IoT hardware and sensor calibration
- 📈 Energy savings analysis for smart lighting systems
